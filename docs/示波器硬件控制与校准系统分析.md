## 一、垂直档位调整时继电器控制流程

### 1.1 整体架构

```
用户操作(UI层/物理按键) → 业务逻辑层 → 硬件抽象层 → 硬件驱动层 → 物理硬件(继电器)
```

### 1.2 触发方式概览

| 触发方式        | 入口文件                          | 消息类型                              | 适用场景      |

#### 1.4.4 ExternalKeysCommand处理档位调整

**文件**: `ExternalKeysCommand.java`

```java
private ExternalKeysMsgVScale msgVScale = new ExternalKeysMsgVScale();

public void moveChannelScale(boolean isAdd, int count) {
    Log.d("TAG", "调整-------------------------m" + isAdd + "c" + count);

    // 获取通道选择组件
    if (channels == null) {
        channels = (MainLayoutCenterChannel) mainViewGroup.findViewById(R.id.mainLayoutCenterChannels);
    }

    // 获取当前选中的通道索引
    int index = channels.getChannelSelectIndex();
    if (index == -1) return;

    // 获取通道对象
    Channel channel = ChannelFactory.getDynamicChannel(index);
    if (channel != null) {
        // 验证档位是否在有效范围内
        double v = channel.getVScaleVal() / channel.getProbeRate();
        if (isAdd && v >= VerticalAxis.getScaleIdValById(channel.getMaxGear())) {
            return;  // 已达到最大档位
        } else if (!isAdd && v <= VerticalAxis.getScaleIdValById(channel.getMinGear())) {
            return;  // 已达到最小档位
        }
    }

    // 构建消息
    msgVScale.setChIndex(index);
    msgVScale.setAdd(isAdd);
    msgVScale.setCount(count);
    
    // 发送RxBus消息
    RxBus.getInstance().post(RxEnum.EXTERNALKEYS_VSCALE, msgVScale);
    
    // 打开右侧滑出菜单
    if (TChan.isChan(TChan.toUiChNo(index))) {
        RxBus.getInstance().post(RxEnum.SLIPRIGHTMENU, MainHolderBottom.SLIPRIGHTMENU_CHANNELS);
    } else if (TChan.isMathToR8(TChan.toUiChNo(index))) {
        RxBus.getInstance().post(RxEnum.SLIPRIGHTMENU, MainHolderBottom.SLIPRIGHTMENU_OTHERS);
    }
    
    // 光标跟踪
    if (ChannelFactory.isChActivate(index)) {
        CursorManage.getInstance().setCursorTrace(true);
        CursorManage.getInstance().curChannelMove();
        CursorManage.getInstance().setCursorTrace(false);
    }
}
```

#### 6. 硬件抽象层 (继电器控制核心)

**这是继电器控制的核心代码**：

```java
// L126-L191: 改变通道垂直档位 - 控制继电器
@Override
public void changeChVolScale() {
    if(!bPowerEnable) return;
    
    long marsk = SHIFTREG_INIT_VAL_MHO10008;  // 移位寄存器初始值
    
    // 遍历所有通道
    for(int ch = ChannelFactory.CH1; ch < chMax; ch++) {
        idx = ChannelFactory.CH8 - ch;
        channel = ChannelFactory.getDynamicChannel(ch);
        resistanceType = channel.getResistanceType();
        
        // 根据当前档位值计算继电器档位索引
        dang = getRatioIdx(resistanceType, channel.getVScaleVal() / channel.getProbeRate());
        
        // 根据档位设置继电器控制位
        switch (dang) {
            case HW.RATIO_DANG_1:  // 第一档
                pdVal |= (0x6L << ((ch * 8)));
                break;
            case HW.RATIO_DANG_2:  // 第二档
                pdVal |= (0xDL << ((ch * 8)));
                break;
            case HW.RATIO_DANG_3:  // 第三档
                pdVal |= (0xBL << ((ch * 8)));
                break;
        }
        
        // 根据阻抗类型(1MΩ/50Ω)设置不同的继电器控制位
        if(resistanceType == Channel.RESISTANCE_50) {
            // 50Ω阻抗的继电器控制
            marsk &= ~(1L << (idx * 8 + 3));
            marsk &= ~(1L << (idx * 8 + 4));
            // ...
        } else {
            // 1MΩ阻抗的继电器控制
            marsk &= ~(1L << (idx * 8 + 2));
            // ...
        }
    }
    
    // 发送移位寄存器命令 - 控制继电器动作
    sendShiftRegister(temp & marsk);  // 第一次发送
    usleep(3000);                      // 延时3ms
    sendShiftRegister(temp);           // 第二次发送（确认）
    
    // 发送PD值到FPGA
    FPGACommand.getInstance().SendChPD_v2(pdVal);
}
```

### 1.4 关键点总结

1. **继电器控制时机**：当垂直档位跨越不同档位区间时触发继电器切换
2. **继电器控制方式**：通过移位寄存器发送64位控制字
3. **延时要求**：继电器动作需要3ms延时后再次确认
4. **阻抗影响**：1MΩ和50Ω阻抗使用不同的继电器控制逻辑
5. **档位区间**：系统将15个垂直档位划分为3个继电器档位区间

## 二、档位与继电器状态对应关系

### 2.1 档位分组定义

根据 `HW_MHO68V2.java` 中的定义：

```java
// 档位区间阈值定义
public static double [] RATIO_DANG_VAL = {0.025, 0.6, 10};      // 1MΩ阻抗
public static double [] RATIO_DANG_VAL_50 = {0.015, 0.1, 1};    // 50Ω阻抗

// 档位索引计算
public int getRatioIdx(int resistanceType, double v) {
    double [] ratioDangVal = resistanceType == 0 ? RATIO_DANG_VAL : RATIO_DANG_VAL_50;
    for(int i=0; i<ratioDangVal.length; i++) {
        if(v <= ratioDangVal[i]) break;
    }
    return i;
}
```

```java
    // 获取阻抗寄存器
    FPGAReg_Resistance regResistance = 
        (FPGAReg_Resistance) getFPGAReg(fpgaIdx, FPGAReg.FPGA_CH_RESISTANCE);

    int val;
    int beginIdx = beginChIdx(fpgaIdx);

    for (int i = beginIdx; i < endIdx; i++) {
        channel = ChannelFactory.getDynamicChannel(i);
        if (channel != null) {
            val = channel.getResistanceType();  // 获取阻抗类型
            regResistance.setResistance(i - beginIdx, val);  // 设置通道阻抗

        }

    TriggerCommon triggerCommon = TriggerFactory.getInstance().getTriggerCommon();
    regResistance.setExtTrigger(triggerCommon.getExtTriggerInputRes());

}
```

### 6.6 关键点总结

1. **双路径处理**：阻抗切换同时通过Channel层和Command层两条路径处理
2. **继电器控制**：阻抗切换会触发继电器动作，通过移位寄存器发送64位控制字
3. **档位联动**：切换到50Ω时，某些档位不可用，会自动切换到有效档位
4. **FPGA同步**：通过FPGA\_CMD\_EXT\_RESISTANCE标志通知FPGA更新阻抗状态
5. **UI联动**：50Ω时AC耦合选项会被禁用

```java
        boolean []ChSample = new boolean[ChannelFactory.CH_CNT];
        int cnt = scope.getChannelSampOnCnt(scope.isRun(true), ChSample);

        if(syncHeader.isSegmentTimestamp()){
            recvSegmentTimestamp(syncHeader, byteBuffer, cnt);
        } else if(syncHeader.isOnlyWave()){
            recvOnlyWave(syncHeader, byteBuffer);
        } else {
            if(syncHeader.isSerialTxtValid()){
                recvSerialTxt(syncHeader, byteBuffer, cnt);
            } else {
                if (syncHeader.isSerialValid()) {

                }
                syncHeader.setIdx(syncHeader.getIdx() + SERIAL_SIZE);
                if (syncHeader.isWaveValid()) {

// L268-L306: 波形数据接收
private void recvWave(SyncHeader syncHeader, ByteBuffer byteBuffer, int cnt, boolean [] ChSample){
    Channel channel;
    int waveLength = syncHeader.getWaveLength();
    boolean bWave = false;

    int beginIdx = FPGACommand.beginChIdx(syncHeader.getFpgaIdx());
    int endIdx = FPGACommand.endChIdx(syncHeader.getFpgaIdx());

    List<Future<?>> listFutures = new ArrayList<>();
    for(int i = beginIdx; i < endIdx; i++){
        if(ChSample[i]){
            channel = ChannelFactory.getDynamicChannel(i);
            if(channel != null && (channel.isNeedWave() || !syncHeader.isSample() 
                    || SyncHeader.isCalibrate() || syncHeader.isLastSample())) {
                // 提交波形处理任务到线程池
                listFutures.add(executorService.submit(new WaveTask(channel, ChSample, cnt,
                        byteBuffer, syncHeader.getIdx(), waveLength, syncHeader)));
                bWave = true;
            }
            syncHeader.setIdx(syncHeader.getIdx() + CH_WAVE_SIZE);
        }
    }
    
    // 等待所有任务完成
    if(bWave){
        for(Future<?> f : listFutures){
            f.get();
        }
    }

    if(bWave) {
        EventFactory.sendEvent(EventFactory.EVENT_CH_WAVE_UPDATE, true);
    }
}
```

```java
    // 触发类型
    public static final int SPI_TRIGGER_FRAME_CS = 1;      // CS帧触发
    public static final int SPI_TRIGGER_FRAME_DATA = 2;    // 数据帧触发
    public static final int SPI_TRIGGER_FRAME_X_DATA = 3;  // X数据帧触发
    
    // 时钟采样边沿
    public static final int SPI_CLK_RISE_EDGE = 0;         // 上升沿采样
    public static final int SPI_CLK_FALL_EDGE = 1;         // 下降沿采样
    
    private int clkChIdx = 0;          // 时钟通道索引
    private int dataChIdx = 1;         // 数据通道索引
    private int csChIdx = 2;           // CS片选通道索引
    private int bits = 8;              // 数据位宽 (4/8/16/24/32)
    private int clkSample = SPI_CLK_RISE_EDGE;  // 时钟采样边沿
    private int dataIdleLevel = IDLE_LEVEL_HIGH; // 数据空闲电平
    private boolean csValid = false;   // CS是否有效
    private int csIdleLevel = IDLE_LEVEL_HIGH;  // CS空闲电平
    private int triggerType = SPI_TRIGGER_FRAME_CS; // 触发类型
    private int triggerMask = 0;       // 触发掩码
```

### 8.9 关键代码分析

#### 8.9.1 SPI配置核心代码

**FPGAReg\_BUS\_SPI.java** - FPGA SPI寄存器配置

```java
@Override
public void configBus(int fpgaIdx, IBus iBus) {
    SpiBus bus = (SpiBus)iBus;
    int triggerType = bus.getTriggerType();
    int clk = bus.getClkChIdx();
    int data = bus.getDataChIdx();
    int cs = bus.getCsChIdx();
    
    // 设置信号源
    FPGAReg_BUS_LEVEL busLevel = (FPGAReg_BUS_LEVEL) FPGACommand.getReg(fpgaIdx, FPGA_BUS_LEVEL);
    setCS(chIdx2BusSrc(cs));
    setSCK(chIdx2BusSrc(clk));
    setDSrc(chIdx2BusSrc(data));
    busLevel.setBusSrcExt(getBusIdx(), ((cs>>2)&0x1) | ((clk>>1)&0x2) | (data&0x4));
    
    // 设置数据位宽
    switch (bus.getBits()){
        case 4:  setBW(0); break;
        case 8:  setBW(1); break;
        case 16: setBW(2); break;
        case 24: setBW(3); break;
        case 32: setBW(4); break;
        default: setBW(1); break;
    }
    
    // 设置CS使能
    if(bus.isCsValid()) {
        setLevel(busLevel, cs, bus.getCsIdleLevel() == IDLE_LEVEL_HIGH ? IDLE_LEVEL_LOW : IDLE_LEVEL_HIGH);
        setEanbleCS(1);
    } else {
        setEanbleCS(0);
    }
    
    // 设置触发模式
    if(SPI_TRIGGER_FRAME_X_DATA == triggerType) {
        triggerType = SPI_TRIGGER_FRAME_DATA;
        setTriggerMask(0);
        setTriggerVal(0);
    } else {
        setTriggerMask(bus.getTriggerMask());
        setTriggerVal(bus.getTriggerData());
    }
    setTriggerMode(triggerType);
    
    // 设置电平
    setLevel(busLevel, clk, bus.getClkSample() == SPI_CLK_FALL_EDGE ? IDLE_LEVEL_LOW : IDLE_LEVEL_HIGH);
    setLevel(busLevel, data, bus.getDataIdleLevel());
    
    FPGACommand.sendCmd(busLevel);
}
```

#### 8.9.2 SPI设备初始化

```java
private void init(){
    SpiPort spiPort = null;
    
    // 初始化各SPI设备
    if(HardwareProduct.isFSpiBoot()){
        mSpiDev[SPI_DEV_FPGA_BOOT] = new FPGA_BOOT_FSpiDev(mContext);
    } else {
        mSpiDev[SPI_DEV_FPGA_BOOT] = new FPGA_BOOT_SpiDev();
    }
    mSpiDev[SPI_DEV_FPGA1_CMD] = new FPGA_CMD_SpiDev();
    mSpiDev[SPI_DEV_FPGA2_CMD] = new FPGA2_CMD_SpiDev();
    mSpiDev[SPI_DEV_CLK] = new NullSpiDev();
    
    // 打开SPI端口
    for(int i=0; i<SPI_DEV_MAX; i++){
        if(mSpiDev[i] != null && !mSpiDev[i].isFSpiDev()) {
            spiPort = mSpiManager.openSpiPort(mSpiDev[i].getDevName());
            if(spiPort != null) {
                mSpiDev[i].onInitDev(spiPort);
            }
        }
    }
}
```
