# 逻辑触发器数据流程分析文档

> **用户配置示例**
> - 四个通道触发条件：都选择"高"（HIGH）
> - 逻辑关系：AND
> - 触发条件：`<`（小于）
> - 时间：100ms

## 五、isFromEventBus参数说明

```mermaid
flowchart TB
    subgraph UserOp["用户操作"]
        U1[用户点击UI控件]
        U2[isFromEventBus = false]
        U3[发送硬件命令]
    end

    subgraph HwSync["硬件同步"]
        H1[FPGA参数变更]
        H2[EventBus事件通知]
        H3[isFromEventBus = true]
        H4[只更新UI，不发送硬件命令]
    end

    U1 --> U2 --> U3
    H1 --> H2 --> H3 --> H4

    style UserOp fill:#dae8fc,stroke:#6c8ebf
    style HwSync fill:#e1d5e7,stroke:#9673a6
```

| 参数值 | 来源 | 行为 |
|--------|------|------|
| `false` | 用户操作 | 发送硬件命令到FPGA |
| `true` | 硬件同步 | 不发送硬件命令（避免循环），只更新UI |

## 七、完整配置后的FPGA触发逻辑

```mermaid

    subgraph Logic["逻辑运算"]

    subgraph Action["触发动作"]
        A1["触发波形采集"]
    end

    C1 --> L1
    C2 --> L1
    C3 --> L1
    C4 --> L1
    L1 --> L2
    L2 -->|满足条件| A1

    style Condition fill:#dae8fc,stroke:#6c8ebf
    style Logic fill:#e1d5e7,stroke:#9673a6
```

**触发逻辑表达式：**
```
当 (CH1=高 AND CH2=高 AND CH3=高 AND CH4=高) AND (持续时间 < 100ms) 时，触发波形采集
```
